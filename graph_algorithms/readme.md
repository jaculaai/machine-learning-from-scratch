# 📊 Graph Algorithms Pro Toolkit

Zbiór interaktywnych algorytmów grafowych z profesjonalną wizualizacją **Dark Mode**. Projekt stworzony w celach edukacyjnych, pozwalający zrozumieć działanie klasycznych algorytmów teorii grafów poprzez porównanie stanu grafu "Przed" i "Po".

## 🚀 Co znajdziesz w środku?

| Algorytm | Opis | Zastosowanie |
| :--- | :--- | :--- |
| **Dijkstra** | Najkrótsza droga w grafie o wagach dodatnich. | GPS, mapy, nawigacja. |
| **Bellman-Ford** | Najkrótsza droga, obsługuje wagi ujemne. | Systemy finansowe, routing sieciowy. |
| **Kruskal** | Minimalne Drzewo Rozpinające (MST). | Projektowanie tanich sieci (kable, drogi). |
| **Prim** | Minimalne Drzewo Rozpinające (metoda budowy). | Rozszerzanie istniejącej infrastruktury. |
| **BFS** | Przeszukiwanie wszerz (fala). | Analiza sieci społecznościowych, labirynty. |
| **Fleury** | Znajdowanie cykli Eulera. | Logistyka, problem rysowania bez odrywania ręki. |
| **Najtańsze Łącze** | Zachłanne rozwiązanie problemu komiwojażera. | Podstawowa optymalizacja tras dostaw. |
